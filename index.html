Figma Personal Access Token은 설정 탭에서 입력해주세요. 브라우저에서 Figma API로 직접 요청하며, Figma가 CORS를 허용하므로 별도 서버가 필요 없습니다.

API 키

API 키는 상단 입력창에서 설정하며, 이 브라우저의 localStorage에만 저장됩니다. 서버로 전송되지 않고 Anthropic API에 브라우저에서 직접 요청합니다 (개인용 도구이므로 이 정도 보안 수준으로 충분합니다).

Figma 연동

Figma 계정 설정(Settings) > Security 탭에서 Personal Access Token을 발급받아 입력하세요. 이 토큰은 브라우저 localStorage에만 저장되며, 룰셋의 "레퍼런스 디자인 추가"와 QA 리뷰어의 "피그마 URL로 검토"에서 사용됩니다.
